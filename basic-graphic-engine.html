Basic Graphics Engine

A module graphics engine that diplays a space scene.

C++ VFX

Overview

This was a project that I was assigned at Full Sail University. The purpose was to make a small graphics engine that displayed certain things that you can pick from a excel sheet. I decided to make a space scene with planets, stars, and a spaceship.

What I Built

  • Direct3D 11 initialization and render loop scaffold
  • Camera movement and rotation and zoom controls
  • Project structure intended for iterative graphics feature additions

Controls

  • WASD — move camera
  • Arrow keys — rotate camera
  • Z / X — zoom in/out

Build & Run

  • Windows and Visual Studio
  • Windows SDK installed
  • Open solution, build, run

(This project is Windows-only because D3D11 is a Microsoft API.)

Technical Notes

  • Focus: real time rendering fundamentals and engine style scaffolding
  • Emphasis on clean iteration loops for adding new graphics features
