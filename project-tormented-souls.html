Tormented Souls

Midterm game project built with a 5-person team. Focused on shipping a playable build and iterating quickly as a group.

C# Unity ShaderLab / HLSL Team Project

Overview

Tormented Souls was a midterm team project where we collaborated across roles to deliver a complete playable experience. My focus was on implementing reliable gameplay systems and integrating features cleanly with the rest of the team’s work.

My Contributions

  • Gameplay: shooting, journal system, player interactions
  • Systems: UI, looting
  • Integration: bug fixes, prefab/scene consistency, performance cleanup

What We Shipped

  • Complete playable build
  • Core gameplay loop and win/lose states
  • UI flow (menu -> gameplay -> results)
  • Audio/FX/polish pass

Technical Notes

  • Primary implementation in C# (Unity)
  • Shader work present (ShaderLab/HLSL) that was used for visuals/presentation
  • Focused on maintainable, team friendly code organization
