Core Programming

  • C# - game tools and editor workflows
  • C++ - systems/engine fundamentals and performance aware coding
  • JavaScript - lightweight tools, UI scripting, web dashboards/portfolio

Game Development Tooling

  • Unreal Engine - Editor Utility workflows, asset validation, JSON reporting
  • Unity - C# gameplay/editor foundations and team project experience
  • Asset standards - naming, folder conventions, LOD/collision expectations, export readiness

VFX / Pipeline Tooling

  • Maya - scene validation and asset preflight tooling
  • 3ds Max - scene cleanup and batch-oriented prep concepts
  • Flow Production Tracking (ShotGrid) - publishing workflows, Version creation, uploads
  • OpenUSD (pxr) - dependency inspection, packaging, portable asset handoffs

Pipeline Patterns I Use

  • Batch operations - folder level processing and summary outputs
  • Structured reporting - JSON/HTML outputs designed for review and automation
  • Artist friendly UX - clear feedback, predictable behavior, minimal friction

Frameworks & Libraries

  • Pillow - image/texture inspection and metadata validation
  • Git - version control and collaborative iteration
