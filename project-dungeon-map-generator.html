Dungeon Map Generator

A Java desktop tool for generating random dungeon maps for tabletop RPGs and game design.

Java Swing Procedural Generation Game Tools

Overview

Dungeon Map Generator is a Java Swing desktop tool I built to create random dungeon layouts for tabletop RPGs, dungeon crawler prototypes, and game design experimentation. The project started as a simple console-based map generator and grew into an interactive editor with themes, visual rendering, generation settings, PNG export, and a Windows build script.

What I Built

  • Procedural dungeon generation with rooms, hallways, doors, enemies, treasure, traps, keys, and exits
  • Multiple room shapes, including rectangle, L-shaped, T-shaped, and cross-shaped rooms
  • Main path, side path, and dead-end generation for more intentional dungeon flow
  • Theme support for changing the visual style of generated maps
  • Interactive Java Swing UI with adjustable generation settings
  • PNG export for saving generated dungeon maps
  • Windows build script for creating a runnable desktop app

Controls & Features

  • Generate Dungeon — creates a new dungeon using the selected settings
  • Regenerate From Seed — recreates the same dungeon layout from the current seed
  • Export PNG — saves the current dungeon map as an image
  • Theme Selector — changes the map color palette
  • Map Size Presets — supports preset map dimensions for different dungeon sizes

Build & Run

  • Built with Java 21
  • Uses Java Swing for the desktop interface
  • Run from source using javac and java
  • Includes a Windows batch file for building a packaged app with jpackage

The packaged Windows version can be launched as a standalone desktop application.

Technical Notes

  • Focus: procedural generation, editor-style UI design, and game tool development
  • Uses seed-based generation so dungeon layouts can be recreated
  • Hallways use pathing logic to reduce awkward room intersections
  • Room placement and generation rules were iterated on to create denser, more usable layouts
  • The project was cleaned up and commented for readability as a finished portfolio piece
